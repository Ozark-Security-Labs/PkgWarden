# Code of Conduct

This project is committed to a welcoming, respectful, and practical community.

## Expected behavior

- Be kind and direct.
- Assume good intent while addressing concrete impact.
- Keep discussion focused on the work.
- Respect maintainers' decisions about scope, safety, and project direction.

## Unacceptable behavior

Harassment, abusive comments, threats, discriminatory language, personal attacks,
and sustained disruption are not acceptable.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or restrict
participation when needed to protect the project.

Report serious concerns privately using the contact path in [SECURITY.md](SECURITY.md).
